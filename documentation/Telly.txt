# niet-builtins ----> voer je uit met execve. 

De locaties genoemd op PATH (env) hierin staan alle niet-builtins. De shell kijkt op deze locaties voor executables van niet builtins.
En kan deze vervolgens uitvoeren met execve:

execve(bin/<command>, char **...., envp);
